# 🎹 Professional Virtual Piano

A high-quality, feature-rich virtual piano with 4 full octaves, professional sound synthesis, and advanced controls.

## ✨ Features

### 🎵 Sound Engine
- **4 Waveforms**: Sine, Square, Sawtooth, Triangle
- **ADSR Envelope**: Full Attack, Decay, Sustain, Release controls
- **Low-pass Filter**: Rich, warm sound
- **Real-time Parameter Adjustment**: Change settings while playing

### 🎹 Keyboard
- **4 Full Octaves**: C3 to C7 (49 keys total)
- **Visual Feedback**: Keys light up when pressed
- **Mouse & Keyboard Support**: Play with mouse clicks or computer keyboard
- **Sustain Pedal**: Hold notes with realistic sustain behavior

### 🎼 Music Features
- **Demo Song**: Plays "Twinkle Twinkle Little Star" + "Für Elise" excerpt
- **Volume Control**: Adjustable master volume
- **Professional Layout**: Realistic piano key positioning

## 🎮 Controls

### Keyboard Mapping
```
Octave 3: Z X C V B N M     (Black keys: S D G H J)
Octave 4: A S D F G H J     (Black keys: W E T Y U)
Octave 5: K L ; ' ] \ =     (Black keys: O P [ ] \)
Octave 6: Q W E R T Y U     (Black keys: 2 3 5 6 7)
```

### Sound Controls
- **Waveform**: Select from 4 different oscillator types
- **Volume**: Master volume control (0-100%)
- **Attack**: How quickly notes reach full volume (0-1s)
- **Decay**: How quickly notes drop to sustain level (0-1s)
- **Sustain**: Volume level while key is held (0-100%)
- **Release**: How quickly notes fade when released (0-2s)
- **Sustain Pedal**: Hold notes even after releasing keys

3. Adjust sound parameters in real-time
4. Try the demo song to hear the piano in action
5. Experiment with different waveforms and ADSR settings

## 🎵 Sound Design Tips

- **Piano Sound**: Use Sine wave with fast attack, medium decay
- **Organ Sound**: Square wave with slow attack
- **Synth Lead**: Sawtooth with fast attack, long release
- **Pad Sound**: Triangle with slow attack, high sustain
